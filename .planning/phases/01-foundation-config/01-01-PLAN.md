  <action>
Create `workspace/synapse_config.py`. The existing `workspace/config.py` is the tools-server config (SERVER_PORT, GROQ_API_KEY, etc.) — do NOT modify it. The new SynapseConfig module lives at `workspace/synapse_config.py` and is imported as `from synapse_config import SynapseConfig`.

Implement the following exactly:

1. `_DEFAULT_SYNAPSE_HOME = Path.home() / ".synapse"` — module-level constant only.

2. `@dataclass(frozen=True) class SynapseConfig:` with fields:
   - `data_root: Path`
   - `db_dir: Path`  (= data_root / "workspace" / "db")
   - `sbs_dir: Path`  (= data_root / "workspace" / "sci_fi_dashboard" / "synapse_data")
   - `log_dir: Path`  (= data_root / "logs")
   - `providers: dict = field(default_factory=dict)`
   - `channels: dict = field(default_factory=dict)`

3. `@classmethod def load(cls) -> "SynapseConfig":` — three-layer precedence:
   - Layer 1: resolve `data_root` from `SYNAPSE_HOME` env var (call `resolve_data_root()`)
   - Layer 2: read `synapse.json` from `data_root` if present (call `_verify_permissions()` first)
   - Layer 3: defaults (empty dicts for providers/channels)
   - Return frozen dataclass with derived paths (`db_dir`, `sbs_dir`, `log_dir` derived from `data_root`, not from file)

4. `def resolve_data_root() -> Path:` — validates SYNAPSE_HOME if set:
   - `raw = os.environ.get("SYNAPSE_HOME", "").strip()`
   - If set: `p = Path(raw).expanduser().resolve()`, then `p.mkdir(parents=True, exist_ok=True)` inside try/except PermissionError — raise RuntimeError with message: `"SYNAPSE_HOME={raw} cannot be created: {e}\nFix the path or unset SYNAPSE_HOME to use the default (~/.synapse/)"`
   - If not set: return `Path.home() / ".synapse"`

5. `def _verify_permissions(path: Path) -> None:` — warn if world/group readable:
   - `mode = path.stat().st_mode`
   - If `mode & (stat.S_IRGRP | stat.S_IROTH)`: `warnings.warn(f"{path} is readable by group/other (mode {oct(mode)}). Run: chmod 600 {path}")`

6. `def write_config(data_root: Path, config: dict) -> None:` — atomic write with mode 600:
   - `config_file = data_root / "synapse.json"`
   - `data_root.mkdir(parents=True, exist_ok=True)`
   - Write to `tmp = config_file.with_suffix(".json.tmp")`
   - `fd = os.open(str(tmp), os.O_WRONLY | os.O_CREAT | os.O_TRUNC, 0o600)`
   - `with os.fdopen(fd, "w") as f: json.dump(config, f, indent=2)`
   - On exception: `os.unlink(tmp)` then re-raise
   - `os.replace(str(tmp), str(config_file))`  — atomic on POSIX
   - `os.chmod(str(config_file), 0o600)`  — MUST re-enforce after replace (umask drift)

Imports needed: `import json, os, stat, warnings` from stdlib; `from dataclasses import dataclass, field`; `from pathlib import Path`; `from typing import Any`.

Do NOT import from `config.py` (tools server). Do NOT use pydantic. Do NOT use keyring.

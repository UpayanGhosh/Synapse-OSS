**Module-level constants:**
```python
DATABASES = [
    "memory.db",
    "knowledge_graph.db",
    "emotional_trajectory.db",
]
```

**Helper functions (all private with _ prefix):**

`_port_open(port: int) -> bool`:
- `socket.create_connection(("127.0.0.1", port), timeout=0.5)` returns True if connects, False on OSError

`_sha256(path: Path) -> str`:
- Streaming SHA-256: read 65536-byte chunks, return hex digest

`_checkpoint_and_close(db_path: Path) -> None`:
- Connect with `sqlite3.connect(str(db_path))`
- Execute `PRAGMA wal_checkpoint(TRUNCATE)` and fetch result
- If `result[0] > result[1]`: raise RuntimeError with message: `"WAL checkpoint incomplete for {db_path.name}: {result[0]} pages in WAL, only {result[1]} checkpointed. Is another process writing to the database?"`
- Always close connection in finally block

**Main function:**
`migrate(source_root: Path, dest_root: Path, dry_run: bool = False) -> dict`:

Step 0 — Port guard:
```python
if _port_open(8000):
    raise RuntimeError(
        "API gateway is running on port 8000. Stop it before migrating.\n"
        "Run: synapse_stop.sh  (or press Ctrl+C in the uvicorn terminal)"
    )
```

Step 1 — Compute source paths and collect existing files:
- `source_db_dir = source_root / "workspace" / "db"`
- For each db_name in DATABASES, for each suffix in `("", "-wal", "-shm")`: add to files_to_copy if exists
- If source_db_dir does not exist or none of the primary .db files exist: raise RuntimeError with clear message: `"Source database directory not found: {source_db_dir}\nIs ~/.openclaw/workspace/db/ the correct location?"`

Step 2 — Initialize manifest:
```python
manifest = {
    "source": str(source_root),
    "dest": str(dest_root),
    "timestamp": time.strftime("%Y-%m-%dT%H:%M:%SZ", time.gmtime()),
    "databases": [],
    "sbs_profiles": False,
    "dry_run": dry_run,
}
```

**Steps 3-9 are ALL inside a single `with tempfile.TemporaryDirectory() as staging:` block.**
The TemporaryDirectory context manager deletes the staging directory on exit. ALL file operations
(checkpoint, copy, verify, final destination write) must happen while the staging dir still exists.
Only the manifest write (Step 10) is outside the with-block because the manifest records the
completed state and the destination files must already be in place before it is written.

```python
with tempfile.TemporaryDirectory() as staging:
    staging_path = Path(staging)
    (staging_path / "db").mkdir(parents=True, exist_ok=True)

    # Step 3 — WAL checkpoint all databases that exist:
    for db_name in DATABASES:
        db_file = source_db_dir / db_name
        if db_file.exists():
            _checkpoint_and_close(db_file)

    # Step 4 — Copy to staging:
    for src, dst in files_to_copy:
        shutil.copy2(str(src), str(dst))

    # Step 5 — Verify checksums:
    for src, staged in files_to_copy:
        src_hash = _sha256(src)
        dst_hash = _sha256(staged)
        if src_hash != dst_hash:
            raise RuntimeError(
                f"Checksum mismatch for {src.name}: "
                f"source={src_hash[:12]}... staged={dst_hash[:12]}..."
            )
        manifest["databases"].append({
            "file": src.name,
            "src_hash": src_hash,
            "dst_hash": dst_hash,
            "size_bytes": src.stat().st_size,
        })

    # Step 6 — Verify row counts:
    for db_name in DATABASES:
        src_db = source_db_dir / db_name
        staged_db = staging_path / "db" / db_name
        if not src_db.exists() or not staged_db.exists():
            continue
        src_conn = sqlite3.connect(str(src_db))
        dst_conn = sqlite3.connect(str(staged_db))
        try:
            tables = [r[0] for r in src_conn.execute(
                "SELECT name FROM sqlite_master WHERE type='table' AND name NOT LIKE 'sqlite_%'"
            ).fetchall()]
            for table in tables:
                src_count = src_conn.execute(f"SELECT count(*) FROM [{table}]").fetchone()[0]
                dst_count = dst_conn.execute(f"SELECT count(*) FROM [{table}]").fetchone()[0]
                if src_count != dst_count:
                    raise RuntimeError(
                        f"Row count mismatch: {db_name}.{table} "
                        f"— source={src_count} staged={dst_count}"
                    )
        finally:
            src_conn.close()
            dst_conn.close()

    # Step 7 — SBS profiles copy:
    sbs_src = source_root / "workspace" / "sci_fi_dashboard" / "synapse_data"
    if sbs_src.exists():
        sbs_staged = staging_path / "synapse_data"
        shutil.copytree(str(sbs_src), str(sbs_staged))
        manifest["sbs_profiles"] = True

    # Step 8 — If dry_run: return manifest before writing to dest (still inside with-block):
    if dry_run:
        return manifest

    # Step 9 — Write to final destination (INSIDE with-block — staging still valid):
    dest_db_dir = dest_root / "workspace" / "db"
    dest_db_dir.mkdir(parents=True, exist_ok=True)
    for src, staged in files_to_copy:
        final = dest_db_dir / staged.name
        shutil.copy2(str(staged), str(final))

    dest_sbs = dest_root / "workspace" / "sci_fi_dashboard" / "synapse_data"
    if (staging_path / "synapse_data").exists():
        if dest_sbs.exists():
            shutil.rmtree(str(dest_sbs))
        shutil.copytree(str(staging_path / "synapse_data"), str(dest_sbs))

# Step 10 — Write manifest (outside with-block — dest files are now written):
manifest_path = dest_root / "migration_manifest.json"
with open(str(manifest_path), "w") as f:
    json.dump(manifest, f, indent=2)

return manifest
```

**Key design decisions:**

1. **Sync wrapper pattern:** `generate()` is a sync method. Wrap the async `SynapseLLMRouter.call()` using:
   ```python
   import asyncio, concurrent.futures

   def _run_async(coro):
       """Run an async coroutine from sync context without blocking the event loop."""
       try:
           loop = asyncio.get_running_loop()
       except RuntimeError:
           loop = None

       if loop and loop.is_running():
           # We're inside an async context (e.g., called from api_gateway.py handler)
           # Use a thread pool to avoid RuntimeError: This event loop is already running
           with concurrent.futures.ThreadPoolExecutor(max_workers=1) as pool:
               future = pool.submit(asyncio.run, coro)
               return future.result()
       else:
           return asyncio.run(coro)
   ```

2. **SynapseLLMRouter role selection in generate():** Map the LLMRouter call context to a litellm router role. Use the `cloud_models` list to determine if a specific model was requested; otherwise default to "casual". If the model string matches a role in model_mappings, use that role directly. Fallback to Ollama if SynapseLLMRouter raises any exception.

3. **Preserve kimi_model attribute** for backward compat with db/server.py status payload. Set `self.kimi_model` to the first entry of `cloud_models` (same as before).

4. **Remove openclaw references**: Delete `SAFETY_SETTINGS`, `SAFETY_LEVEL`, `google-antigravity/` prefix handling, `_extract_gateway_text()`, and all urllib.request imports.

5. **Preserve embed()**: Leave `embed()` using `ollama.embeddings()` (no change needed).

**New generate() implementation:**
```python
def generate(self, prompt, system_prompt="You are a helpful assistant.", force_kimi=False) -> str:
    """
    Route prompt through SynapseLLMRouter (litellm) with Ollama fallback.
    force_kimi arg preserved for backward compat but ignored.
    """
    messages = [
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": prompt},
    ]

    # Determine role from cloud_models (first entry) or default to "casual"
    role = "casual"
    if self.cloud_models:
        candidate = self.cloud_models[0]
        # If cloud_models contains a synapse role name (not a model string), use it directly
        cfg = self._synapse_config.model_mappings if self._synapse_config else {}
        if candidate in cfg:
            role = candidate

    try:
        text = _run_async(self._synapse_router.call(role, messages, max_tokens=1024))
        if text:
            return self._sanitize(text)
    except Exception as exc:
        logger.warning("SynapseLLMRouter failed for role '%s': %s", role, exc)

    # Ollama fallback
    return self._sanitize(self._call_ollama(prompt, system_prompt))
```

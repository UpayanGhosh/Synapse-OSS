# Day 2 — The problem: I needed an AI that actually knows me

## Script (90 seconds)

I got tired of starting over every single session.

Every AI tool I tried did the same thing — completely blank at the start of each conversation. No memory of how you speak, what you care about, what you told it last week. You'd spend the first ten minutes re-explaining yourself before you could get anything useful done.

I speak Banglish. Bengali written in English letters, mixed with actual English. Most models treat that as a fringe case. For me it's just how I talk. The assistant I use every day needs to handle that without asking me to clarify what I mean.

And beyond language — I wanted something that builds a model of me over time. My writing patterns. The topics I return to. How I communicate when I'm tired versus when I'm in flow.

So I built Soul-Brain Sync. Every message updates a realtime state immediately. Every 50 messages — or every 6 hours, whichever hits first — a batch process runs and distills all of it into 8 profile layers: core_identity, linguistic patterns, emotional state, domain knowledge, interaction style, vocabulary, exemplars, meta context.

That profile compiles into a 1500-token segment injected into every system prompt.

It took three rewrites before it stopped drifting into noise. But it works now.

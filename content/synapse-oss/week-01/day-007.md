# Day 7 — Synapse in one diagram: the full system overview

## Script (90 seconds)

I built this for 12 months. Here's the map.

11 subsystems. 4 messaging channels. 6-plus LLM providers. All running on one laptop.

Start from the outside. Messages come in from WhatsApp, Telegram, Discord, or Slack. Each channel has its own handler — a class implementing the same 6-method interface. They all plug into a shared pipeline.

That pipeline is the gateway layer: FloodGate buffers rapid messages, dedup filters repeats, a task queue serializes work, two workers process tasks concurrently.

Each worker pulls context from three places. The Soul-Brain Sync profile — 8 JSON layers that capture who I am and how I communicate. The hybrid memory store — vector search and full-text search merged and reranked. And the dual cognition engine, which generates an internal monologue before forming the final response.

All of that feeds into the LLM router. Which provider actually runs depends on the message type: Banglish and general queries go to Gemini Flash, code questions go to Claude with thinking mode on, anything private routes only to local Ollama.

The response comes back through whichever channel the message came from.

The knowledge graph sits alongside memory — storing relationships between concepts, not just raw conversation text.

That's the whole thing. This is day 7. Week 1 done.

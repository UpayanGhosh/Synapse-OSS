# Demo profile

`synapse.demo.json` (repo root) ships a zero-friction profile for the five-minute demo path: it uses `fastembed` for embeddings (no Ollama download, no GPU), routes every LLM role through OpenRouter's free `google/gemini-2.0-flash-exp:free` model, and disables the dual-cognition engine to keep token spend predictable. Real users should copy this file to `~/.synapse/synapse.json` and customize providers, model mappings, and session toggles to taste — set `OPENROUTER_API_KEY` (or supply an optional `GEMINI_API_KEY` to switch the routes to native Gemini) before running `docker compose -f docker-compose.demo.yml up`. The project's JSON loader (`workspace/synapse_config.py`) uses stdlib `json.load()` which rejects `//` and `/* */` comments, so the demo profile is plain JSON; an inline `_comment` key carries a short pointer back to this file.
